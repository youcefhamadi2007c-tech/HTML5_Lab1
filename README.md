# Light Yagami Story - HTML Project

## Description
This project is an HTML fan project about **Light Yagami** from the anime and manga *Death Note*. 
It tells the story of Light discovering the Death Note, his rise as Kira, and the intellectual battle with the detective **L**. 
The project includes sections for the story description, key characters, important events, and famous quotes. 

## Features
- Simple and clean HTML layout.
- Fully self-contained: no CSS or JavaScript required.
- Easy to read and share on GitHub or any web browser.
- Suitable for fan projects or learning HTML structure.
